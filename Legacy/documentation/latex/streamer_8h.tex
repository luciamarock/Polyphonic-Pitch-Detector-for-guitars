{\footnotesize\ttfamily int chunker (\begin{DoxyParamCaption}\item[{int}]{buffdim,  }\item[{int}]{elemcount }\end{DoxyParamCaption})}

This function basically divides the total length (in number of samples) of the audio file in input by the length of the chosen samples buffer, in this case 256, this way we obtain the number of buffers contained in the audio file. ~\newline
 There is of course a bunch of remaining samples this can be ignored, in fact they are not returned, the calculation was made just for educational purposes because it is done calling another function
